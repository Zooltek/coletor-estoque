# RFC-0000

## Design System

### Objetivo

Criar um Design System para padronizar toda a interface do projeto.

Este documento é a única referência visual do projeto.

Toda implementação futura deve seguir este padrão.

---

# Regras

* Não alterar regras de negócio.
* Não alterar funcionalidades.
* Não criar novos padrões visuais.
* Todo componente deve seguir este documento.

---

# Diretórios

Criar:

```text
src/
    theme/
        colors.js
        spacing.js
        typography.js
        radius.js
        shadows.js

components/
    common/
```

---
